### map {#map}

#### 移除 map中某个元素

内置函数 delete

```
delete(map,key)
```

#### 清空 map

内置函数 clear(go 1.21 以后) 或者逐一 delete

```
clear(map)
```

#### 查找某个元素

这样写是错的，遍历是 n 级复杂度

```
for i, _ := range map {
		if i == map[val] {
			return false
		}
	}
return true
```

这样写 O(1)

```
if v,exist :=  map[val]; exist {
	return exist
}
```
